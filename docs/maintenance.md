# Manutenção

Este repositório inclui scripts de manutenção para:

- limpeza de logs e recursos antigos (`scripts/cleanup.sh`)
- backups automatizados (`scripts/backup.sh`)
- rollback de deploy (`scripts/rollback.sh`)
- relatórios de saúde e monitoramento (`scripts/health-monitor.sh`)

Use `./scripts/health-monitor.sh --report` para gerar um resumo das métricas recentes.
